SMS Verification - The Easy Way!

Perfect Solution: SMS goes directly to Google Voice without triggering CallBunker authentication!

1 Go Back to Twilio Verification

Return to the Twilio verification popup where you tried the phone call:

Open Twilio Verified Caller IDs
Click "Add a new Caller ID" and enter +16174921250 again

2 Look for SMS Option

In the verification popup, look for one of these options:

Common SMS Button Names:
  • "Send text message instead"
  • "Text me"
  • "SMS verification"
  • "Send SMS"
Instead of:
  • ❌ "Call me now"
  • ❌ "Voice verification"
  • ❌ "Phone call"

3 Check Google Voice for SMS

After clicking the SMS option:

  1. Open Google Voice app or voice.google.com
  2. Look for a new text message from a Twilio number
  3. The message will contain a 6-digit verification code
  4. Copy that code
Why This Works: SMS doesn't use call forwarding, so CallBunker never sees it!

4 Enter Code in Twilio

Back in the Twilio verification popup:

  1. Enter the 6-digit code from the SMS
  2. Click "Verify" or "Submit"
  3. Wait for confirmation
Success! Your Google Voice number is now verified for CallBunker Dialer outbound calls!
Ready to Test CallBunker Dialer!

Once verification is complete, return to your dashboard and try the dialer.

Your outbound calls will show +1 (617) 942-1250 as caller ID!
